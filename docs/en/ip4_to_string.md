| | |
|:---|:---|
| **Input	IP4** | BOOL (string that contains the IP address) |
| **Output** | DWORD (decoded IP v4 address) |
| | IP4_TO_STRING converts the IP4 address stored as DWORD in a string. The format is  'NNN.NNN.NNN.NNN'. |
