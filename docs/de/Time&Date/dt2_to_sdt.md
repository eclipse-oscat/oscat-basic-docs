## Type	Funktion : SDT

| | |
|:---|:---|
| **Input	DI** | DATE (Datum) |
| **TI** | TOD (Tageszeit) |
| **Output** | SDT (Strukturierter Datums Zeit Wert vom Typ SDT) |
| | DT2_TO_SDT wandelt ein Datum und eine Tageszeit um in eine strukturierte Datum Tageszeit vom Typ SDT. |
